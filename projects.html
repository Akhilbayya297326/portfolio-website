Krishi Chakra 2.0

The Neural Matrix of Agriculture

"While the world discusses AI for Silicon Valley, we looked at the backbone of our nation. Most agri-tech apps today are just digital noticeboards—they don't think, predict, or protect. Krishi Chakra transforms the traditional farmer into a data-driven enterprise manager."

Smart AgriCore & Commercial Gateway

  • Subscription Gate: A scalable revenue model. Users unlock the 'Pro Matrix' (Gemini AI features) via dynamic backend authentication (e.g., Activation Code: JUDGE-PASS).
  • Geospatial Telemetry: Automatically locks onto the farmer's satellite location, cross-referencing local groundwater tables and regional soil heuristics.
  • The Business Matrix: Calculates Expected Revenue vs. Net Profit per acre based on live Mandi trends.

Active Care & Resource Sharing

  • Pest Radar: Uses a Haversine Distance Algorithm to map nearby threats, triggering Multilingual Voice Alerts within a 10km radius.
  • Hydro Smart: A Life-Cycle Irrigation Matrix that saves up to 50% groundwater by adjusting to specific crop stages.
  • Sahayog (Agri-Uber): A secure booking engine for machinery sharing, generating Unified UPI QR codes for transparent P2P transactions.

Technical Backbone (The Secret Sauce)

Built with a mobile-first React.js frontend. Intelligence is powered by Google Gemini AI 1.5 Flash for rapid, multimodal analysis. The backend runs on Node.js/Express with a dual-engine CORS policy, connected to a real-time MongoDB Atlas Cloud database for instant field-to-admin syncing.

Java Spring Boot + MVC

MediFlow AI

Intelligent Hospital Management System

"Traditional systems are highly fragmented. Patients struggle to find specialists, beds are tracked on static clipboards, and legacy software lacks backend security. MediFlow bridges the gap between public accessibility and enterprise-grade administrative control."

Public Gateway & Live Tracking

  • Symptom-Based AI Routing: Patients bypass medical jargon. Typing "severe chest pain" dynamically filters the DB to match a Cardiologist and securely generates a token.
  • Live Resource Tracking (CRUD): Dynamic DB tracking 100+ hospital beds. Doctors update patient status (Pre-op, Critical) which instantly refreshes public capacity metrics.

Enterprise Security (RBAC)

  • Role-Based Access: System intercepts logins, routing ADMINS to the Command Center and DOCTORS to the Clinical Dashboard.
  • Encrypted Credentials: Utilizes spring-boot-starter-security. Passwords are never plain-text; they are encrypted using BCryptPasswordEncoder.
  • Fail-Safe Protection: The SecurityFilterChain instantly intercepts unauthorized endpoint access and redirects to the secure login gateway.

Pure Java Engineering Stack

Engineered purely in Java Spring Boot utilizing a strict MVC architecture (Controller-Service-Repository pattern). Data is persisted via MySQL mapped by Spring Data JPA/Hibernate. The frontend is cleanly rendered server-side using the Thymeleaf Templating Engine with Bootstrap 5.
